# Insurance Claims Processing System

A robust backend system for processing insurance claims built with Node.js, TypeScript, Express, and MongoDB.

## Features

- Client management (create, retrieve clients)
- Policy management (create policies for clients)
- Claim submission and processing
- Advanced MongoDB aggregations for reporting
- Input validation using Joi
- Comprehensive error handling
- Unit tests with Jest
- TypeScript for type safety

## Prerequisites

- Node.js (v14 or higher)
- MongoDB (v4.4 or higher)
- npm or yarn
